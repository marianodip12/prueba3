# 🧤 HB Analytics — Goalkeeper Pro

Aplicación web gratuita para análisis avanzado de arqueros de handball.

## 🚀 Deploy en Vercel (desde GitHub)

1. Subí esta carpeta a un repositorio GitHub
2. Importá el repo en [vercel.com/new](https://vercel.com/new)
3. Vercel detecta Vite automáticamente — no toques nada
4. Hacé clic en **Deploy** ✅

## 💻 Correr en local

```bash
npm install
npm run dev
```

Abrí `http://localhost:5173`

## 🛠 Stack

- React 18 + Vite 5
- Recharts (gráficos)
- LocalStorage (persistencia, sin backend)

## 📦 Módulos

| Módulo | Descripción |
|--------|-------------|
| 🏆 Partidos | Crear / editar partidos y porteros |
| ⚡ Registrar | Registro de cada lanzamiento |
| 📊 Equipo | Stats globales + evolución temporal |
| 🧤 Porteros | Stats individuales + comparativa |
| 🎯 Zonas | Heatmap de portería 3×3 |
| 📈 Avanzado | Stats por origen, técnica, sistema, etc. |
| 🧠 IA | Análisis automático + alertas tácticas |
